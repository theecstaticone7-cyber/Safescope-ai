# Models

Trained ML model files. Generated by running `ai-service/ml/train_model.py`.

## Files
| File | What It Is |
|---|---|
| risk_classifier.pkl | XGBoost classifier → predicts Low / Medium / High risk for an area+time |
| incident_regressor.pkl | XGBoost regressor → predicts expected incident count for an area+time |
| vector_store/ | FAISS index files for the RAG pipeline (generated by ai-service/rag/vector_store.py) |

## Do NOT commit .pkl files to Git
Add `models/*.pkl` and `models/vector_store/` to `.gitignore`.
These are generated artifacts — regenerate them by running the training script.
